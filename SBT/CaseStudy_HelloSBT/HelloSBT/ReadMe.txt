This is a simple case study to understand SBT build process.

This project covers..
	- Scala, Java sourcecode
	- Scala, Java test cases
	- build.sbt, plugin.sbt, build.properties
	- JUnit and ScalaTest dependency added as for Java and Scala testing
	
Usage : 

Use following SBT basic commands to generate and publish build to your local Maven repository

>sbt
>clean
>compile
>test
>run
>publish (In case you want the artifacts to be published to your local Maven repository)
>publishLocal (In case you want the artifacts to be published to Ivy)
